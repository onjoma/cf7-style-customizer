## â“ FAQ

<details>
<summary><b>Does this plugin work with all Contact Form 7 forms?</b></summary>

Yes! The styles are applied globally to all Contact Form 7 forms on your site automatically.

</details>

<details>
<summary><b>Can I use custom CSS in addition to this plugin?</b></summary>

Absolutely! You can add custom CSS to further customize your forms. The plugin's styles use `!important` to ensure they take precedence over default CF7 styles, but you can override them with more specific selectors.

</details>

<details>
<summary><b>Will my settings be lost if I deactivate the plugin?</b></summary>

No, your settings are safely stored in the WordPress database and will be preserved if you deactivate and reactivate the plugin.

</details>

<details>
<summary><b>Can I style different forms differently?</b></summary>

Currently, the plugin applies styles globally to all CF7 forms. Individual form styling may be added in future versions.

</details>

<details>
<summary><b>Is the plugin translation-ready?</b></summary>

Yes! The plugin is fully internationalized and ready for translation.

</details>

<details>
<summary><b>Does it affect site performance?</b></summary>

No! The plugin generates minimal inline CSS only on pages with Contact Form 7 forms, ensuring optimal performance.

</details>
